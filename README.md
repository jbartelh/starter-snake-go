## battlesnake-go

Battlesnake-AI written in Go.

Forked from [battlesnakeio/starter-snake-go](https://github.com/battlesnakeio/starter-snake-go).

### run

Switch to working directory and run:
```
go run .
```

If there are missing packages:

```
go get -t ./...
```
